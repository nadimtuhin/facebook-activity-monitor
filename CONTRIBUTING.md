# Contributing

## Getting Started

1. Fork the repository
2. Clone your fork: `git clone https://github.com/YOUR_USERNAME/facebook-activity-monitor`
3. Install dependencies: `npm install`
4. Start dev server: `npm start`

## Development

- Source code lives in `src/`
- `npm start` — starts webpack-dev-server with HMR
- `npm run build` — production build

## Submitting Changes

1. Create a feature branch: `git checkout -b feature/my-change`
2. Make changes and commit with a clear message
3. Push to your fork and open a Pull Request against `main`

## Reporting Bugs

Open a [GitHub Issue](https://github.com/nadimtuhin/facebook-activity-monitor/issues) with:
- Browser and extension version
- Steps to reproduce
- Expected vs actual behaviour

## Code Style

The project uses `.editorconfig` and Babel. Keep changes consistent with the existing style.
